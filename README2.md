**System-Sentinel** is a powerful, real-time system monitoring tool built with Python. It provides a visual, terminal-based dashboard to track your CPU, RAM, Disk, and Network usage instantly.

---

## 🔥 Key Features

* **Real-time Updates:** Refreshing every second to give you live hardware stats.
* **Visual Gauges:** Beautiful terminal-based progress bars for CPU and RAM.
* **Color-coded Alerts:** Changes color when your system is under high load (Green -> Red).
* **Lightweight:** Minimal impact on system performance.

---
